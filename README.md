# Module Name: Data Structures 101

A collection of artisanal, hand-crafted data structures implementing fundamental computer science concepts from scratch in Python.

## Features
- **Linear Data Structures**: Implementation of Singly Linked Lists, Doubly Linked Lists, Stacks, and Queues.
- **Non-Linear Data Structures**: Implementation of Binary Search Trees and Hash Maps with separate chaining.

## Learning Objectives
- Implement complex logic without relying on third-party magic.
- Understand the underlying mechanics of memory management and node-based structures.
- Analyze performance characteristics using real-world benchmarks and complexity analysis.

## Project Structure
- `linked_list.py`: Singly Linked List implementation.
- `doubly_linked_list.py`: Doubly Linked List implementation.
- `stack.py`: Stack (backed by Linked List).
- `queue_ds.py`: Queue (backed by Doubly Linked List).
- `bst.py`: Binary Search Tree implementation.
- `hash_map.py`: Hash Map with separate chaining.
- `COMPLEXITY.md`: Summary of Big O time and space complexities.

## Requirements
- Python 3.10+

## How to Run
Run the demo script to see data structures in action:
```bash
python main.py
```

## Testing
Run all tests using the built-in unittest module:
```bash
python -m unittest discover tests
```
